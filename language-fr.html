2. Structure des données des primitives

2.1 Constructeurs

mot expr expr
(mot expr …)
Concatène deux (ou plusieurs) mots en un seul
montre (mot "a "b "c)
liste chose1 chose2
(list chose1 chose2 …)
Crée une nouvelle liste à partir des données indiquées
montre (liste 1+2 2+3 3+4)
phrase chose1 chose2
(phrase chose1 chose2 …)
ph chose1 chose2
(ph chose1 chose2 …)
Crée une nouvelle liste à partir des données (si ce ne sont pas des listes) ou des éléments de listes
montre phrase [1 2 3] [et ainsi de suite]
pajoute chose liste
dajoute chose liste
Donne la liste avec ajout de chose comme premier/dernier élément ; si liste est un mot, concatène (fusionne).
montre pajoute 0 [ 1 2 3 ]
montre pajoute "a "bcd
montre dajoute 4 [ 1 2 3 ]
montre dajoute "e "bcd
tableau taille
(tableau taille origine)
Crée un nouveau tableau. L'origine par défaut est 1.
montre tableau 10
tableaumd [dimensions …]
(tableaumd [dimensions …] origine)
Crée un nouveau tableau multidimensionnel. L'origine par défaut est 1.
montre tableaumd [2 2]
listeentableau liste
(listeentableau liste origine)
Crée un nouveau tableau à partir d'éléments d'une liste
montre listeentableau [ 1 3 5 7 ]
tableauenliste tableau
Crée une nouvelle liste à partir d'éléments d'un tableau
montre tableauenliste { 2 4 6 8 }
combine chose1 chose2
Si chose2 est un mot, comme mot; sinon, comme pajout
montre combine "a [b c]
inverse liste
Donne une liste avec l'ordre inversé des éléments ; si la donnée est un mot, inverse les carctères
montre inverse [ 1 2 3 ]
montre inverse "abc
symgen
Donne une chaîne unique, par exemple G1, G2, etc…
montre symgen

prendsdebut nomdepile

prendsfin nomdepile

fais "q "ab ajoutefin "q "c montre :q montre prendsfin "q

Teste si chose est une liste ou une chaine vide.
egalp expr expr
egal? expr expr
expr = expr
pasegalp expr expr

Test d'égalité ou d'inégalité. Compare des chaines, des nombres ou des listes (égales si le nombre d'éléments et les éléments sont égaux).
avantp chose1 chose2

Teste si les choses ont la même identité.

souschainep chose1 chose2

Teste si chose1 est une sous-chaine de chose2.

4. Arithmétique

4.1 Opérations numériques

Les données entrées sont des nombres ou des expressions numériques, la donnée en sortie est un nombre.

expr - expr

(produit expr …)
expr * expr
quotient expr expr
(quotient expr)
expr / expr
puissance expr expr
expr ^ expr
Respectivement, ajoute, soustrait, multiplie, divise et mets à la puissance de. Une entrée seule à quotient donne le réciproque.
reste expr expr
expr % expr

Donne le reste (modulus). Pour reste et %, le résultat a le même signe que la première donnée ; pour modulo le résultat a le même signe que la seconde donnée.

7. Gestion de l'espace de travail

7.1 Définition de procédure

cree nomproc données … déclarations … fin/
Définie une nouvelle procédure nommée. Les données peuvent être :
  • Requis : :a :b
  • Facultatif (avec valeurs par défaut) : [:c 5] [:d 7]
  • Reste (données restantes comme liste) : [:r]
  • Nombre de données par défaut : 3
cree étoile :n repete 5 [ av :n td 144 ] fin
definis nomproc [[données …][déclarations …]]
Défini une nouvelle procédure nommée avec des données facultatives.
definis "étoile [[n][repete 5 [av :n td 144]]]
def nomproc
Donne la définition d'une procédure nommée comme une chaine.

7.2 Définition de variable

fais nomvar expr
Mes à jour une variable ou défini une nouvelle variable globale. Le nom de la variable doit être précédé d'un guillemet.
fais "mavar 5
nom expr nomvar
Comme fais mais avec les données inversées.
nom 5 "mavar
locale nomvar
(locale nomvar …)
le fais suivant créera la variable dans le domaine local au lieu du domaine global.
locale "mavar
faislocale nomvar expr
Défini une variable dans le domaine local (raccourci pour locale puis fais)
faislocale "mavar 5
chose nomvar
Donne la valeur d'une variable. :foo est un raccourci pour chose "foo.
montre chose "mavar
globale nomvar
Prévoit la variable dans le domaine global. Ceci ne fait rien de bien utile.
globale "mavar

8. Structures de contrôle

8.1 Contrôle

execute [ déclarations … ]
Exécute les déclarations spécifiées une fois.
exécute [ av 100 td 90 ]
executeresultat [ déclarations … ]
Exécute les déclarations spécifiées une fois. Si les déclarations sortent ue valeur, le résultat est une liste comportant la valeur comme seul élément. Sinon, le résultat est une liste vide.
repete expr [ déclarations … ]
Repète les déclarations expr fois.
repete 4 [ av 100 td 90 ]
boucle [ déclarations … ]
Repète les déclarations en boucle. Utilisé dans une procédure définie par l'utilisateur qui se termine par sortie, stop ou salut
boucle [ fais "n aleatoire 100 montre :n si :n == 0 [ salut ] ]
nombrerep
#
Donne l'itération actuelle de repete ou boucle
repete 10 [ montre nombrerep ]
repete 10 [ montre # ]
si expr [ déclarations … ]
si [expr] [ déclarations … ]
Exécute les déclarations si l'expression est non-nulle.
si 2 > 1 [ montre "ouaip ]
sisinon expr [ déclarations … ] [ déclarations … ]
sisinon [expr] [ déclarations … ] [ déclarations … ]
Exécute le premier jeu de déclarations si l'expression est non-nulle, sinon, exécute le deuxième jeu de déclarations.
sisinon 1 > 2 [ montre "ouaip ] [ montre "nop ]
teste expr
teste [expr]
Teste l'expression spécifiée, enregistre le résultat dans le domaine local pour utilisation ultérieur par sivrai ou sifaux
sivrai [ déclarations …]
siv [ déclarations …]
sifaux [ déclarations …]
sif [ déclarations …]
Exécute les déclarations si le résultat de la dernière procédure teste locale était non-nulle (vrai) ou nulle (faux), respectivement.
teste 1 > 2 sivrai [ montre "ouaip ] sifaux [ montre "nop ]
stop
Interrompt l'exécution de la procédure sans valeur en sortie.
sortie expr
so expr
Interrompt la procédure lancée et donne la valeur spécifiée.
intercepte motcle listeinstructions
Exécute les instructions mais si une erreur avec motcle correspondant survient, donner la valeur (s'il y en a une). Utiliser "ERROR pour intercepter les erreurs des procédures habituelles.
intercepte "t [ montre "avant indique "t montre "apres ]
intercepte "erreur [ montre 1 / 0 ] montre erreur
indique motcle
(indique motcle valeur)
Indique une erreur avec le mot-clé indiqué qui peut-être interceptée. Une valeur de sortie facultative peut être donnée.
montre intercepte "t [ montre "hello (indique "t "mot) ]
erreur
Donne une liste décrivant la dernière erreur interceptée : un numéro d'erreur, un message d'erreur et un nom de procédure où l'erreur s'est produite.
intercepte "erreur [ montre 1 / 0 ] montre erreur
attends duree
Suspend l'exécution. duree est en 60e de secondes.
salut
Interrompt le programme.
.sortieconditionnelle expr
Comme sortie si expr donne une valeur, sinon comme stop.
ignore expr
Evalue et ignore les résultats de l'expression.
fais "q [ 1 2 3 ] ignore prendsfin "q
` liste
Donne la liste avec substitutions :
  • , listeinstructions remplacé par la sortie
  • ,@ listeinstructions remplacé par les éléments de liste donnés
  • ",instruction remplacé par la sortie prefixée par "
  • ,@instruction remplacé par la sortie prefixée par :
montre `[a b ,[sp [c d e]] f ,@[sp [g h i]]]
fais "v "x montre `[",:v :,:v]
pour listedecontrole [ déclarations …]
Boucle for typique. La listedecontrole spécifie trois de quatre éléments : le nom de la variable locale, la valeur de debut, la valeur limite et la taille d'incrementation facultativement.
pour [ a 1 10 ] [ montre :a ]
pour [ a 0 20 2 ] [ montre :a ]
faisxfois [nomvar nbfois] [ déclarations …
Exécute les déclarations le nombre de fois spécifié dans nbfois. Le nomvar est défini à l'actuel nombre d'itération.
faisxfois [ i 5 ] [ montre :i * :i ]
fais.tantque [ déclarations …] expr
fais.tantque [ déclarations …] [expr]
Exécute les déclarations spécifiées au moins une fois et les répète tant que l'expression est non-nulle (vrai).
fais.tantque [ fais "a aleatoire 10 montre :a ] :a < 8
tantque expr [ déclarations …]
tantque [expr] [ déclarations …]
Exécute les déclarations spécifiées seulement tant que l'expression reste non-nulle (vrai).
tantque (aleatoire 2) = 0 [ montre "zéro ] montre "un
fais.jusque [ déclarations …] expr
fais.jusque [ déclarations …] [expr]
Exécute les déclarations spécifiées au moins une fois et les répète tant que l'expression est nulle (faux).
fais.jusque [ fais "a aleatoire 10 montre :a ] :a < 8
jusque expr [ déclarations …]
jusque [expr] [ déclarations …]
Exécute les déclarations spécifiées uniquement tant que l'expression reste nulle (faux).
jusque (aleatoire 2) = 0 [ montre "un ] montre "zéro
cas valeur [ propositions … ]
Pour chaque proposition dans l'ordre : Si la proposition est de la forme [ SINON expr ] alors expr est évaluée et donnée. Sinon, Si la proposition est de la forme [ [ correspondances ] expr ] et valeur est un élément de correspondances alors expr est évaluée et donnée.
montre cas :var [ [ ["a] "AAA ] [ ["b] "BBB ] [ sinon "autre ] ]
cond [ propositions … ]
Pour chaque proposition dans l'ordre : Si la proposition est de la forme [ SINON expr ] alors expr est évaluée et donnée. Sinon, Si la proposition est de la forme [ [ expr1 ] expr2 ] et expr1 est évaluée comme étant non-nulle (vrai) alors expr2 est évaluée et donnée.
montre cond [ [ [:var = 1] "un] [ [:var = 2] "deux] [ sinon "autre ] ]

8.2 Itération basée sur template

Ces procédures de niveau supérieur sont uniquement compatible avec la forme des procédures nommées. La première donnée est le nom d'une procédure à appeler.

applique nomproc liste
Appelle nomproc avec comme données les éléments de liste.
invoque nomproc entree1
(invoque nomproc entree1 …)
Appelle nomproc avec les données spécifiées.
invoque "montre "salut
pourchaque liste nomproc
Appelle nomproc pour chaque élément de la liste.
pourchaque [1 2 3] "montre
appliqueliste nomproc liste
Donne une liste composée des résultats de l'appel de nomproc pour chaque élément de la liste.
pour double :n sortie :n * 2 fin montre appliqueliste "double [1 2 3]
filtre nomproc liste
Donne une liste composée de la liste indiquée où l'application de nomproc sur l'élément donne un résultat non-nul (vrai).
pour oddp :n sortie binet :n 1 fin montre filtre "oddp [ 1 2 3 4 5 ]
recherche nomproc liste
Donne le premier élément dans liste pour lequel l'application de nomproc donne un résultat non-nul (vrai). Si rien n'est trouvé, la liste vide est donnée.
pour oddp :n sortie binet :n 1 fin montre recherche "oddp [ 2 8 4 7 10 9 ]
reduis nomproc liste
(reduis nomproc liste initial)
Appelle nomproc de manière répétée avec deux données - la valeur actuelle et l'élément de liste suivant. Si initial n'est pas spécifié, le premier élément de liste sera utilisé à la place.
montre reduis "somme [ 1 2 3 4 5 ]
appliquecroiseliste nomproc listeliste
(appliquecroiseliste nomproc liste liste …)
Appelle nomproc de manière répétée avec les données des listes dans toutes les combinaisons possibles.
montre appliquecroiseliste "mot [[a b] [x y]]
montre (appliquecroiseliste "mot [a b c] [1 2 3 4])
